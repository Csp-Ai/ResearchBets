# ResearchBets Infrastructure (`infra`)

## Responsibility
`infra` stores deployment, environment, and infrastructure-as-code documentation/scripts for the ResearchBets platform.

## Interfaces
- Defines environment conventions consumed by `apps/web` and all `services/*` modules.
- Contains provisioning and deployment references aligned to Layer 1, Layer 2, and Layer 3 services.
- Documents runtime dependencies, secrets boundaries, and operational procedures.

## Ownership
- **Primary owner:** DevOps / Infrastructure Team
- **Contributors:** Security Engineering, Service Owners
