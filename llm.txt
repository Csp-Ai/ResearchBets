## Operating Rules for LLMs/Codex
- Preserve spine continuity on all internal nav actions by using `nervous.toHref()` with `appendQuery()`; avoid manual query-string concatenation.
- Keep demo/live language truthful and emotionally neutral.

  - Avoid alarmed wording.
- Never scrape prohibited sportsbooks/platforms (including FanDuel/PrizePicks); prefer official providers/APIs and deterministic demo fallbacks.
- Performance rule: do not pull heavy dependencies into first render when avoidable; lazy-load OCR and larger diagnostics/widgets behind user intent.
- Keep dynamic import boundaries around non-critical panels on critical routes (`/ingest`, `/stress-test`, `/control`) to protect first paint.
- Do not add new alternate landing/today variants without a deprecation or archive plan.
- If you modify routes or APIs, update README route mapping in the same change.
